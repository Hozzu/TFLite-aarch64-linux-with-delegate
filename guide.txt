All build guide is based on tensorflow r2.3 source code.

1. Tensorflow Lite for aarch64 linux
Build with following command:
bazel build -c opt --config=elinux_aarch64 //tensorflow/lite:libtensorflowlite.so


2. GPU delegate for aarch64 linux
1) Modify all following lines in bazel build for gpu delegate (tensorflow/tensorflow/lite/delegates/gpu/BUILD, tensorflow/tensorflow/lite/delegates/gpu/cl/BUILD)
"//tensorflow:android" --> "//tensorflow:linux_aarch64"

2) Add egl and opengles library path to cross compiler (tensorflow/third_party/toolchains/embedded/arm-linux/cc_config.bzl.tpl)
if (ctx.attr.cpu == "aarch64"):
    default_link_flags_feature = feature(
        name = "default_link_flags",
        enabled = True,
        flag_sets = [
            flag_set(
                actions = all_link_actions,
                flag_groups = [
                    flag_group(
                        flags = [
                            "-L/YOUR_PATH",
                            "-lstdc++",
                            "-Wl,-z,relro,-z,now",
                            "-no-canonical-prefixes",
                            "-pass-exit-codes",
                            "-Wl,--build-id=md5",
                            "-Wl,--hash-style=gnu",
                        ],
                    ),
                ],
                ),

3) Build with following command:
bazel build -c opt --config=elinux_aarch64 //tensorflow/lite/delegates/gpu:libtensorflowlite_gpu_delegate.so


3. Hexagon delegate for aarch64 linux build command
1) Add following lines into bazel build file for hexagon delegate (tensorflow/tensorflow/lite/delegates/hexagon/BUILD):
cc_binary(
    name = "libtensorflowlite_hexagon_delegate.so",
    linkopts = [
        "-Wl,-soname=libtensorflowlite_hexagon_delegate.so",
    ],
    linkshared = 1,
    linkstatic = 1,
    tags = [
        "nobuilder",
        "notap",
    ],
    deps = [":hexagon_delegate"],
)

2) Build with following command:
bazel build -c opt --config=elinux_aarch64 //tensorflow/lite/delegates/hexagon:libtensorflowlite_hexagon_delegate.so

3) Ask for hexagon-TFLite interface library(libhexagon_interface.so) for non-Android aarch64 linux on tensorflow github
